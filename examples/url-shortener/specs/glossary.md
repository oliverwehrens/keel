# Glossary — ubiquitous language

| Term       | Definition                                                      | Synonyms to avoid     |
| ---------- | -------------------------------------------------------------- | --------------------- |
| Long URL   | The original, full-length URL a user wants to shorten.         | "target", "real url"  |
| Short code | The compact token that stands in for a long URL.               | "slug", "hash", "id"  |
| Link       | The mapping between one short code and one long URL.           | "record", "entry"     |
| Resolve    | Look up the long URL for a given short code.                   | "expand", "decode"    |
